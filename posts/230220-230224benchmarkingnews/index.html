智能汽车新功能收集23.02.20-23.02.24

智己L7

  1. ISC智能车灯交互场景： ISC智能交互灯组是由全车前后共5000颗LED单元构成，通过高集成度的独立LED像素点控制，显示图形与动画效果
    • 智能车语系统里可以提供宠物迎宾的模式，宠物会在车主上车前迎接主人的到来
    • 车主走向车辆时，ISC灯组能够显示出车辆的当前电量。而在车辆进行充电时，智能车语系统会在ISC灯组对充电过程进行动画展示，并以比例图像展现目前电量状态
    • 启行人模式后，车头的ISC灯组将展示动态的行人标识。同时车尾的ISC灯组也会同步显示“行人过马路”的动效，告知后方车辆
    • 当行人正在通行时，智能车语系统会与智驾的感知系统协作，实时监测后方来车。如果有车辆正在靠近时，ISC会呈现黄色并闪烁，以此来警示行人，防止“鬼探头”事故的发生
    • 除了ISC自带的艺术纹理皮肤、个性化迎宾外，厂家在云端更新Icon、皮肤、灯效，用户可通过手机下载到车端，用户还可通过IMOS或手机APP，编辑车语等内容
  2. DLP辅助场景： DLP辅助场景主要通过投影技术将画面投射至车头前方，提示驾驶者及行人
    • 示宽光毯通过整车感知系统，自动识别前方车道收窄，DLP可投射与车同宽的光毯，帮助用户参考前方通行难度，地判断可通过性
    • 光线较差的夜间场景中，红外摄像头可自动识别行人和骑行车辆，智能车语系统及时通过DLP投射光束来定位行人、骑行车辆的位置，并在前方地面上投射行人图标，提醒驾驶者注意
    • 车辆高速行驶中，如果与前车跟车距离过近时间超过3-5秒时，智能车语系统就会向前方路面投影出雷达纹，提醒驾驶者注意保持车距。系统根据不同驾驶者的驾驶习惯，提供了两档灵敏度的选择
    • 夜间专注驾驶时，智能车语系统通过在路面投射指向的箭头，让驾驶者可以始终目视前方，降低了夜间驾驶的疲劳度，提升驾驶通过复杂路口的安全性
    • 用户可以通过OTA进行更新更新导航Icon式样
  3. Onehit场景化体验
    • 超跑模式：39英寸联屏会下降至一半的高度，UI界面会切换至超跑主题，加速时的播放超跑声浪效果，CDC、加速制动响应、氛围灯等配合联动
    • 可以组合成自己专属的模式，可以自定义语音唤醒来执行模式：自定义一键迎接女主，内置小憩模式，一键调节座椅、空调、香氛，屏幕更会显示由teamLab定制的繁花动效
    • 宠物模式，系统自动调整空调通风，车头和车尾ISC可温馨表达出宠物车内状态
  4. 智能座舱及IMOS操作系统:
    • 品牌宣传：但智己汽车则以“上善若水”为灵感，把人、功能、生态、服务都串联在一起，提出了一种全新的交互模式，称为 HAI （Human Auto Interaction），也就是希望利用人和车打造一种共生、共享的环境。
    • 原石系统：原石获取的途径则是用户贡献驾驶数据，这些数据能够帮助智己汽车为后续的产品进行升级、改善，原石可做作为智己商城的代币进行消费
    • “无界”车机交互：可在中枢屏上进行歌曲的操作和选择，同时在长联屏上看到歌词显示，也可以分享给到副驾，提供视觉上的流动
    • IMOS空中领航：通过智能驾驶与高精地图的数据融合，可提前洞悉超视距，预知前方全部路况
    • 车载 Carlog：摄影家马良共创大师模版，可编辑，一键处理视频的背景、环境，生成沉浸AR效果，支持Carlog成片一键上传，分享至IM APP、抖音等社交平台

哪吒S

  1. 中控屏调试座椅

    • 可在中控屏上直接调试前排和后排座椅角度,

  2. 查看车胎胎压和行驶动画

    • 主屏最大的内容给了导航地图，在主屏右上角悬浮了一个小窗，可以查看车辆的胎压和行驶时的动画，点进去是经过动画渲染的车辆模型。

  3. 快捷菜单和负一屏

    • 下拉的快捷菜单以及左滑呼出负一屏，这个负一屏包括了账户登录以及最近使用的应用、音响模式、情景模式等。中控屏下端是卡片式应用，支持自定义编辑，最下端则是固定的菜单栏，包括主页、空调、车辆、应用库以及座椅调节按钮，点进去就是二级菜单。

  4. 包含众多APP应用

    • 哪吒S不仅支持自带的腾讯视频、雷石KTV等APP，还支持众多小程序，比如宝宝巴士、股票软件等等

  5. 座舱交互

    • 座舱交互层面，语音交互依然是哪吒 S 的核心能力，此外还有眼球追踪、手势交互。
    • 车内配备了 DMS（驾驶员监测）摄像头以及 OMS（车内监控系统）摄像头，因为这两个传感器的存在，系统可以追踪驾驶员的眼球以及识别车内成员做出的手势。
    • 当驾驶员看向中控屏导航界面时，系统会预测驾驶员有操作需求，这个时候语音系统会处在免唤醒的状态待命，准备随时执行驾驶员的语音指令。
    • 在手势交互层面，支持像音量控制、视频的快进快退等简单常规的交互。

  6. 副驾驶屏幕与主驾驶屏操作交互

    • 主要提供了一些影音娱乐功能，还可以和液晶显示屏联动，将副驾驶所选定的地图导航信息投射到液晶仪表盘上。

路特斯的智能座舱

  1. 数字双生概念：外面下雨时，车机里也会“下雨”，数字双生的本质，是还原物理世界。路特斯在车机屏里做了一个数字化的车辆模型，将车辆的诸多控制集中在车机上，用户对着数字车模进行点击，就可以实现对车门、后备箱、充电口等多个部位的控制，还原不仅体现在静态时，接下来当车辆行驶在路上后，车机里的画面也会对真实世界进行还原（动态还原具体程度未知）

    其思路是让车机和真是世界联动，给用户提供充满精致感和趣味性的体验

  2. 路特斯在智能座舱上的研发重点：
    • 车控。比如打开天窗、车门，用户会在这类场景下用语音替代双手；
    • 音乐。路特斯希望用户在搜歌曲时，不仅搜到中英文的音乐时系统可以识别，甚至在搜索小众乐队时也能找到；
    • 第三，导航。比现在的导航更加极致，当用户搜索一个目的地时，系统不要问第几个，而是要细化到目的地的哪个大门

苹果CarPlay

全新一代CarPlay功能包括：

  1. 可向车辆内的多块显示屏发送内容，包含：多媒体中控屏、数字仪表盘、娱乐显示屏等。
  2. CarPlay车载还可获取车辆数据，包括：无缝显示行驶速度、燃油余量、车内温度及更多的仪表信息。
  3. CarPlay车载直接完成控制收音机、空调等调整车内环境等操作。用户无需退出CarPlay界面，便可操作收音机、空调等车内设备。
  4. 用户将可个性化定制自己的驾驶体验，选择不同的仪表盘显示界面，如：传统三圆仪表、双圆+中央信息显示屏等多种布局。
  5. 新增对小组件的支持，用户将可直接在车辆的仪表盘上查看天气和音乐等App的速览信息。

星途瑶光

  1. 智能场景：
    • 瑶光具备多维识别功能，可根据情绪识别结果主动推送相关歌单，调节用户情绪
    • 根据不同的动静态手势，完成音量调节、歌曲切换等操作
    • 识别用户的视线同时使屏幕亮起，让所见即所控；此外，瑶光
    • 通过DMS摄像头技术，对疲劳、通话、抽烟、分心、睡眠以及音量六大驾乘场景进行监测，自动完成打开车窗或空调、降低音量以及预警等操作，带来更贴心的人性化体验
    • 四音区技术，全车位置实现精准语音操
  2. 千人前面：
    • 瑶光还拥有千人千面模式，通过打造私享场景和自定义场景，让用户自由决定出行场景模式
  3. 超级ID：
    • 瑶光拥有首发科技超级ID功能，可实现“一人多车、一车多人”跨车登陆，在与朋友互换开车时，可切换自己的账号
    • 使用人脸识别、声纹、NFC、蓝牙、机械钥匙等方式均可快速绑定账号，用户无需改变原有的出行习惯
  4. 瑶光还聚合第三方优势生态资源，打造智能生态，具备宝宝模式、生日彩蛋、浪漫彩蛋三大约定模式
  5. 支持语音复刻：
    • 1分钟内即可录入家人的声音，让陪伴如影随形
  6. 车辆解锁：
    • NFC卡片钥匙，轻触后视镜即可解闭锁
    • 手机蓝牙钥匙解锁

上汽零束和OPPO合作

手机与车机打通，上汽和OPPO将在SOA应用服务生态方面开展合作，打造多场景跨终端的应用服务，OPPO应用服务框架将被引入车机端，丰富车端生态。OPPO应用服务开发者平台与上汽SOA开发者平台将实现跨域融合

  1. 将手机与车机连接后，手机APP也将和车机上的应用一起展示在车机桌面上，从车端和手机端都可以进行操作。而且，用户从手机端即可实现对车辆的控制
  2. 在手机上开启小憩模式，车辆就会调整设备，包括放倒座椅，调整灯光，调暗屏幕等。
  3. 将手机连接到车机上玩游戏时，系统可以实现音画与控制分开，即音画都在车机端展示，而手机则可用作手柄进行操控
  4. 智能预测功能是当用户在手机上复制一段文字，如果文字中包含电影名或者音乐名，系统就会自动预测并提问是否需要播放相关电影或音乐，选择确定，车机将直接开始播放
  5. 手机上选择一段地址，车机也可开启导航
  6. 在车-机互联的情况下，如果在车上参加视频会议，当到达目的地下车时，视频可以直接流转到手机，不会影响连续性
  7. XR虚拟现实
    • 蔚来的AR眼镜主要还是用于观影，可以结合车内的音响系统，实现影院般的观影体验
    • AR眼镜作为增强现实，应该更多的是供驾驶员使用，显示仪表盘和显示屏上的内容，包括车速、油耗、导航等功能。目前其AR眼镜功能更接近于VR虚拟现实眼镜，主要还是用于娱乐观影
    • 奇瑞GENE概念车：GENE具备元模式（Meta），进入该模式，方向盘会主动退缩远离驾驶员进入自动驾驶，同时隐藏在方向盘下方的虚拟现实眼镜(VR)会推出。乘客佩戴眼镜后座椅会自动调节到半躺模式，为虚拟体验提供舒适感。交互中心中央的锥体也会同时升起到驾驶员可触及的舒适位置，提供虚拟交互体验。

小鹏P7

  1. 操作界面：与手机的操作模式很是相似，在小鹏P7的中控屏上，从上方通知栏往下滑，就会出现一个快捷键页面，囊括了几种常用的车辆操作。在中控屏左侧也有固定的一竖条快捷按键，是一些常用应用的入口，其逻辑很像手机底部的放置应用程序的快捷栏。从上至下分别是主菜单、车辆模式、自动泊车辅助、空调以及音乐
  2. 车辆模式控制界面: 车辆常用的功能设置都可以在这里找到，可以调节驾驶模式，小鹏P7拥有标准、运动以及经济三种驾驶模式调节，还有能量回收强度的调节。下方还有后备厢、充电口、后视镜、车窗、座椅的调节
  3. 灯带支持音乐随动调节
  4. 空调控制：左右两侧显示的是双温区温度调节，上方是玻璃加热、座椅通风、座椅加热以及风量调节，中间显示的是车内的虚拟影像，拥有可视化的出风口，可以直接通过触屏调节任意角度的出风口方位。下方显示的则是常用出风口位置调节以及空调模式调节
