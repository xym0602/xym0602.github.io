智能汽车新功能收集06.13-06.17

一、高合 HiPhi X

  1. H-SOA超体电子电气架构，让用户可通过中控屏，实现车门、空调、座椅的操控，升空车门、灯光、音乐、香氛联动
  2. HiPhi X的图形化开发自定义场景工具HiPhi Play，可让用户自定义卡片，分享用车生活，千人千乘
  3. 六一联名场景卡，基于HiPhi AIC的创作功能，就将座舱转变为灵感画室；“一键看球”场景卡，让用户在车内，可自由联动空调、空气过滤系统、座椅、氛围灯等，让用户全程于车内五感享受球赛乐趣；可以通过调度车灯，车门，音响系统，从而实现模拟鞭炮或者烟火的效果。
  4. HiPhi X搭载的是全球首个可进化自定义数字灯光系统，由PML可编程智能大灯和ISD智能交互灯两大创新设计的核心灯组。PML可编程智能大灯可以投射斑马线、支持自定义视频投影功能等；ISD智能交互灯可针对用户不同的驾驶场景，提供各种各样的智能化表情包，不仅可以交流互动，也可以用于驾驶过程中起到警示、提醒作用

思考：车内场景联动，场景配置社区分享下载、智能灯交互-表情包

三、智己 L7 智能汽车

  1. 沉浸式智能场景体验。智己L7将可根据路况智能切换模式，自动调整至让用户感觉最舒适的状态；并能通过声、光、电全域联动的超跑模式（包括整车底盘状态、动力体验、制动脚感、人机交互等）
  2. 智己L7创新推出Carlog智能车载摄像系统，拍照并分享
  3. 智己汽车坚定软硬件协同升级的技术路线，将“可升级硬件”作为成长躯干，将“软件迭代”作为发展思维，数字驱动云上迭代，从而让“智能汽车”这一数字生命体，具备了自我进化的能力
  4. 将“原石谷用户数据权益平台”作为核心，构建起全新的生产关系，使智能汽车成为驱动元宇宙不断开拓价值的必要引擎

四、北汽

  1. 首搭华为智能座舱，车机如手机般好用：车机能和华为手机、手表、眼镜等智能终端进行无缝流转，打开车门，视频通话就会无缝连接车机，除了视频之外，音乐、导航、通话等等，都能在车机和智能终端之间无缝切换。
  2. 车机与智能家居进行互联：它不仅能和华为智能终端进行互动，还能与智能家居进行互联，打开回家模式，就能一键控制家里的智能灯光，扫地机器人就会自动开始“工作”，智能空调会为你提供最舒适的温度，踏进家门就会享受到干净温馨；启动离家模式，全屋的灯光会自动关闭，智能窗帘也会缓缓打开，迎接窗外的阳光。
  3. 车机更加智能：不仅能进行声纹识别、面部识别、手势识别，当车里放着音乐而家人睡着时，你只需要把手指放在嘴边，“嘘”一下，它就会自动调低音量；还能进行高难度的唇语识别，当窗外噪音比较大时，会通过你的口型来辨识你的指令。
  4. 把后视镜装进座舱内：雨雪天气开车时，后视镜上经常挂雨滴，很影响视线。为解决这一痛点，”北京汽车 魔方”给出的方案是：把后视镜装进座舱内。它在同级中率先配备了电子外后视镜，不仅能解决雨雪天气和夜间行驶中的视线模糊问题。而且，它的视野更广阔，是传统外后视镜的3倍，当遇到高速、转弯、倒车等情况时，能大幅缩小盲区，从而提高安全性。

五、极狐阿尔法S全新HI版

  1. 拥有21.7寸中控大屏幕，全场景语音交互，常用命令1秒内响应，且不依赖网络，在地下停车场也可语音控车。
  2. 用硬件将智能驾驶带上新高度，新车拥有34个高感知硬件，组成一套360度全覆盖的超高融合感知系统，可真实还原3D物理世界，对障碍物精准识别；MDC810智能驾驶计算平台支持400 TOPS超强算力，可轻松应对复杂路况的考验
  3. 智能泊车功能，当我们来到停车场、或者地下车库停车，车辆就会记录下这些信息，之后我们再来到这里，只需要点击对应的位置，这款车就可以自动去找车位，实现自动泊车。要是那个车位已经有车，那么它就会在附近找一个车位。此外，这款车型也支持跨层泊车。
  4. 配备了Harmony智能座舱系统，鸿蒙车机系统支持车内与不同华为设备间（PC、智能穿戴、汽车屏幕、智慧屏、平板等）的高清视频和高清语音通话，如手机进行的视频通话，可以很轻松地流转到车机上，方便快捷
